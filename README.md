# peer2pair
nwHacks 2023 Project

## Docker
- `cd peer2pair-frontend`
- `docker build . -t peer2pair`
- `docker run -p 3000:3000 peer2pair`

## Frontend
- `npm install`
- `npm start` to develop

## Backend
- `npm install`
- `npm run dev`
- `npm run start`
